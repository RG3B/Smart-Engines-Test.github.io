Smart ID Engine is a cross-platform, offline software for the recognition and detection of document fraud indicators in IDs, documents of title, visas, and other documents used for identification and access control, operating on video streams, photographs, or images.
Smart ID Engine DOES NOT transmit your customers' personal data to third-party services and/or third parties for manual input, DOES NOT store data — all processing is performed in the device's local random-access memory, and DOES NOT require a network connection. When using Smart ID Engine, no additional actions related to obtaining the data subject's consent for the processing of their personal data are required. Smart ID Engine consists of the following modules:

  1. The core library is a binary code designed to run on various computer architectures under different operating systems.
  2. The document recognition configuration is contained in a separate configuration file (referred to as a "bundle file").

The implemented mechanism allows for the initial integration of only the binary code (effectively the library itself) into the client solution, while the required configuration (the necessary bundle file) can be loaded on demand depending on the documents needed.
