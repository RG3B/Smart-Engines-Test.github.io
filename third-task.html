Overview

Productname is a command-line tool for forgery detection in images.
Supported image file formats:

  • .jpg
  • .tif
  • .png

Analysis output

Result of an analysis consists of a JSON file with a report (see Listing 1) and a duplicate image with -forgery-mask.png suffix. Outputting image has the same dimensions as a source image and it has a "mask" — color-highlighted regions the tool marks as suspicious (see pic.1).

Conduct an analysis

Run a console command:

%USERPROFILE%/productname path cipher

  • path — absolute path to the analyzed image;
  • cipher — user's cipher.
